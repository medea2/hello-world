# hello-world
Mit dem Online-Schalter für Kinder wollen wir einen Prototyp entwickeln, der aufzeigt wie ein Dialog zwischen Verwaltung und Kinder aussehen kann. Kinder sind eine Bevölkerungsgruppe, die besonders viele staatliche Dienstleistungen in Anspruch nimmt, jedoch selbst kaum in der Ausgestaltung dieser beteiligt ist. 
Ein Leuchtturmprojekt, um diese Beteiligung zu etablieren, soll illustrieren wie integrative öffentliche Angebote im 21. Jahrhundert aussehen können. Die Kinder machen während des Projekts erste Erfahrungen als gesellschaftliche GestalterInnen und ihr Recht auf Teilhabe in der Gesellschaft wird gestärkt.
